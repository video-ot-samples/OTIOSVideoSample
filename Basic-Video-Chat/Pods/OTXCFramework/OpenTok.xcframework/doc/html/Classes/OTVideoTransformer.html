Creates an instance of a Vonage video transformer.

- (nullable id)initWithName:(NSString *)name_ properties:(NSString *)properties_

Parameters

name

The name of the Vonage transformer. Currently, “BackgroundBlur” is supported.
properties

A JSON string with the properties of the Vonage video transformer. For a background blur transformer, the format of the JSON is “{\"radius\”:\“None\”}“. Valid values for the radius property a None”, “High”, and “Low”.
